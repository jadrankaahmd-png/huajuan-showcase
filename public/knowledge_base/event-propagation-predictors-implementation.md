## 📊 **功能概览**

基于 sim-predict 的社会模拟预测模型，实现了三个事件传导预测器：

---

## 🔍 **功能1：FDA 事件传导预测器**

### **核心功能**

1. **监听 FDA 药物批准/拒绝事件**（通过 NewsAPI）
2. **用多Agent角色传导机制预测股价影响**
3. **预测信息被 price in 需要多长时间**
4. **输出**：方向预测 + 传导时间预测

### **信息传播路径**

```
FDA → 分析师/机构（2小时）→ KOL（1小时）→ 记者（3小时）→ 散户（6小时）
```

**总 Price-in 时间**：12小时

### **基线结果**

- **方向准确率**：66.7%
- **传播曲线匹配**：92.1%
- **Price-in 时间准确率**：34%（需优化）

### **示例输出**

```json
{
  "event": {
    "title": "FDA grants breakthrough therapy designation",
    "publishedAt": "2026-03-10T11:24:08.887Z"
  },
  "prediction": {
    "direction": "上涨",
    "confidence": 0.55,
    "priceInTime": 12
  },
  "tradingWindow": {
    "start": "立即",
    "end": "12小时后",
    "optimalEntry": "前30分钟",
    "optimalExit": "9小时后"
  }
}
```

## 💰 **功能2：财报事件传导预测器**

### **核心功能**

1. **监听美股财报发布事件**（通过 Finnhub）
2. **预测财报发布后信息传播路径**
3. **预测股价何时完全消化信息**
4. **输出**：方向预测 + price-in 时间预测

### **信息传播路径**

```
财报发布 → 分析师（1小时）→ KOL（0.5小时）→ 记者（2小时）→ 散户（4小时）
```

**总 Price-in 时间**：7.5小时

### **示例输出**

```json
{

    "date": "2026-03-17"
  },
  "prediction": {
    "direction": "需要财报数据",

    "start": "财报发布后",
    "end": "7.5小时后",
    "optimalEntry": "前15分钟",
    "optimalExit": "5小时后"
  }
}
```

## 🚀 **功能3：信息滞后机会扫描器**

### **核心功能**

1. **实时扫描市场事件**（通过 NewsAPI）
2. **识别"信息滞后"交易机会**（散户还没反应的窗口）
3. **输出**：机会列表 + 预估剩余时间窗口

### **传播阶段判断**

| 已过时间 | 当前阶段 | 机会分数 | 描述 |
|---------|---------|---------|------|
| 0-1小时 | 分析师/机构阶段 | 0.9 | 最早，机会最大 |
| 1-2小时 | KOL 阶段 | 0.7 | 仍然有机会 |
| 2-5小时 | 记者阶段 | 0.4 | 机会较小 |
| 5+小时 | 散户阶段 | 0.1 | 几乎没有机会 |

### **示例输出**

```json
{
  "totalOpportunities": 1,
  "opportunities": [
    {
      "标题": "Consumer spending remains robust",
      "已过时间": "0.3小时",
      "当前阶段": "分析师/机构阶段",
      "剩余窗口": "4.7小时",
      "机会分数": 0.9
    }
  ]
}
```

## 🛠️ **技术实现**

### **API 数据源**

1. **NewsAPI** - FDA 相关新闻、市场新闻
2. **Finnhub** - 财报日历、市场情绪
3. **QVeris** - 美股实时数据

### **多Agent角色传导模型**

基于 sim-predict 的 5类 Agent 参数：

```yaml
agents:
  analyst:  { count: 4,  influence: 0.9, speed: 0.3, skepticism: 0.7 }
  kol:      { count: 8,  influence: 0.7, speed: 0.9, skepticism: 0.3 }
```
