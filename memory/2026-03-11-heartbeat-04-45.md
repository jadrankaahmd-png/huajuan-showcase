## 💡 关键学习

### **1. Telegram 抓取需要代理**
- 必须使用 HTTP 代理（不是 SOCKS）
- 需要 PySocks 库
- 代理地址：127.0.0.1:8118

### **2. Telegram 首次授权**
- 需要输入验证码
- 验证码会发送到手机
- 授权后生成 session 文件

### **3. Vercel 部署问题**
- Vercel 可能缓存旧代码
- 需要手动触发重新部署
- 或使用 Vercel CLI 强制部署

### **4. Redis 存储**
- Vercel 读不到本地文件
- 必须使用 Redis 存储
- Key: telegram:news:latest
