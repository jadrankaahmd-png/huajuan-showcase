## 🚨 重要发现

### **根本原因：crontab 自动提交覆盖回滚！**

**crontab 中有一个每15分钟自动提交的任务：**
```bash
*/15 * * * * cd ... && git commit -m "auto: update telegram news ..." && git push
```

**这就是为什么每次回滚后会被覆盖！**

**解决方案：**
- ✅ 暂停 crontab
- ⚠️ **如果需要恢复 crontab**，需要注释掉自动提交的行：
  ```bash
  crontab /tmp/crontab_backup.txt
  crontab -e  # 注释掉 git commit && git push 的行
  ```
